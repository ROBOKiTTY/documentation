# Quality of Life Features

This page documents lesser known quality of life features available on Revolt.

## sed Replacement

You can do sed replacement with messages by sending a message in the format as follows:

```
s/{regex}/{replacement}(/{flags})
```

- `{regex}`: any valid regex expression
- `{replacement}`: what to replaced matched substring with
- `{flags}`: optionally, flags to use for the regex
